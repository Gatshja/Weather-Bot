## 📖 เกี่ยวกับโปรเจกต์ (About The Project)

**Weather Bot** คือบอทสำหรับ Discord ที่ถูกพัฒนาขึ้นเพื่อช่วยอำนวยความสะดวกในการเช็คสภาพอากาศ รายงานพยากรณ์อากาศ และแจ้งเตือนฝนตกแบบ Real-time โดยดึงข้อมูลมาจาก OpenWeatherMap API

*(หมายเหตุ: ปัจจุบันระบบทั้งหมดหยุดทำงานแล้ว)*

## ✨ ฟีเจอร์ที่เคยให้บริการ (Key Features)

ฟังก์ชันหลักที่บอทเคยทำได้ ได้แก่:

* 🌤 **เช็คสภาพอากาศปัจจุบัน:** แสดงอุณหภูมิ ความชื้น และสภาพท้องฟ้า
* ☔ **แจ้งเตือนฝนตก:** แจ้งเตือนล่วงหน้าเมื่อมีแนวโน้มฝนจะตกในพื้นที่ที่กำหนด
* 📅 **พยากรณ์อากาศล่วงหน้า:** สรุปสภาพอากาศใน 3-7 วันข้างหน้า
* 📍 **ค้นหาตามพื้นที่:** รองรับการค้นหาสภาพอากาศตามชื่อเมืองหรือพิกัด GPS

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)

* **Language:** Node.js
* **Library:** Discord.js
* **API:** OpenWeatherMap API, Location Static Maps
